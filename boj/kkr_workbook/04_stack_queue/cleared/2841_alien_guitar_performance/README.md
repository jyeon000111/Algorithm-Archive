## 1. 문제 정보 (Problem)

  - **문제 이름**: 외계인의 기타 연주
  - **출처 및 번호**: [백준 2841번]('https://www.acmicpc.net/problem/2841')
  - **난이도**: 실버 1
  - **문제 요약**: 멜로디 연주에 필요한 최소 손가락 움직임 구하기

---
## 2. 핵심 아이디어 (Core Logic)
- 줄 번호를 인덱스로, 현재 누르고 있는 fret 위치를 스택으로 저장하는 이중 리스트를 만든다.
- 스택에는 오름차순으로 프렛이 쌓이도록 한다.

---
## 3. 어려웠던 점 (Difficulties)
- 스택을 담은 이중 리스트를 잘못 만들어서, 조작할 줄이 2개 이상일 때 오류가 생겼다.
    - `[[]] * (N+1)` : 내부 리스트가 같이 변경됨. 다 같은 하나의 리스트를 가리킴
    - `[[] for _ in range(N+1)]` : 독립적인 내부 리스트 생성하는 리스트 컴프리헨션 기법

- 입력 단계에서 시간 초과가 발생했다.
  - 파이썬에서 이런 문제 생기면 아래 코드 맨 위에 넣어보기!
```python
import sys
input = sys.stdin.readline
```

---
## 4. 새롭게 배운 점 (What I Learned)
- input() 과 sys.stdin.readline() 의 차이
  - input() 함수가 상대적으로 느린 이유: 추가 작업(사용자에게 보여줄 프롬프트 처리 등)이 내부적으로 포함되어 있기 때문
  - sys.stdin.readline() 함수가 빠른 이유: 입력 버퍼에서 한 줄을 직접 읽어오는 낮은 수준의 함수기 때문.
    => 대량의 입력 처리할 때 훨씬 빠름!!!

## 2. 핵심 아이디어 (Core Logic)
- 주어진 시간들을 오름차순 정렬해서 누적합을 더하면 됨. (그리디 알고리즘)
    1. 첫번째 방법: 슬라이싱해서 누적합을 더해준다. (매번 새 리스트 생성, 중복된 계산 반복 -> 비효율적???)
    2. 두번째 방법: 각 인덱스의 값이 몇 번 더해질지 헤아린 후, 곱해준다.
  - 두번째 방법으로 진행. i인덱스의 값은 (N - i)번 더해짐.

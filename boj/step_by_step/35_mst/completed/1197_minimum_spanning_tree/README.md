## 1. 문제 정보 (Problem)

  - **문제 이름**: 최소 스패닝 트리
  - **출처 및 번호**: [백준 1197번]('https://www.acmicpc.net/problem/1197')
  - **난이도**: 골드 4
  - **문제 요약**: 최소 스패닝 트리의 가중치 출력

---
## 2. 핵심 아이디어 (Core Logic)
- 간선 정보 저장. 가중치 기준으로 sort
- 정점이 V개면 사이클 없이 간선 V-1개 연결하면 최소 신장 트리가 됨. (종료 조건)
- 가중치 낮은 간선부터 연결 -> 이미 연결되어 있으면 패스. 연결되어 있지 않으면, 연결한 간선 카운트 + 가중치 누적
- 연결 여부는 유니온파인드로 확인!

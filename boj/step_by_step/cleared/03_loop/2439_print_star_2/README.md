## f-string 서식으로 문자열 정렬하기
1. 기본
```python
>>> name = '홍길동'
>>> age = 30
>>> f'나의 이름은 {name}입니다. 나이는 {age}입니다.'
'나의 이름은 홍길동입니다. 나이는 30입니다.'
```

2. 정렬
```python
>>> f'{"hi":<10}'  # 왼쪽 정렬
'hi        '
>>> f'{"hi":>10}'  # 오른쪽 정렬
'        hi'
>>> f'{"hi":^10}'  # 가운데 정렬
'    hi    '

>>> f'{"hi":=^10}' #공백 채우기
# 채워 넣을 문자는 정렬 문자 `<`, `>`, `^` 바로 앞에 넣기
```

# 코드 개선
- 이전 코드
```python
n = int(input())
for i in range(1, n + 1):
    print(" " * (n - i) + "*" * i)
```

- 최종 코드
```python
#f-string 서식

n = int(input())
for i in range(1, n + 1):
    print(f'{"*" * i:>{n}}')
```

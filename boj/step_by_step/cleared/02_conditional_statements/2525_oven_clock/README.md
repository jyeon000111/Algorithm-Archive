#### 1\. 문제 정보 (Problem)

  * **출처 및 번호**: [백준 2525번](https://www.acmicpc.net/problem/2525)


#### 2\. 핵심 아이디어 (Core Logic)

  * **시간과 분을 모두 분으로 변환**해서 계산하는 것이 핵심.
  * 주어진 `시(H)`와 `분(M)`을 `H * 60 + M`으로 변환.
  * 여기에 요리 시간 `T`를 더함.
  * 계산된 총 시간을 24시간 형식에 맞게 다시 시와 분으로 변환.
      * 최종 **시** = `(총 분 // 60) % 24`
      * 최종 **분** = `총 분 % 60`


#### 3\. 코드 (Code)

```python
h, m = map(int, input().split())
time = int(input())

total_minutes = h * 60 + m + time

h_final = (total_minutes // 60) % 24
m_final = total_minutes % 60

print(h_final, m_final)
```


#### 4\. 회고 (Retrospective)

  * **시간 복잡도**: `O(1)`. 입력 크기와 상관없이 연산 횟수가 일정하다.
  * **새롭게 배운 점**: 시간 계산 시 `if`문 대신 \*\*나머지 연산자(`%`)\*\*를 사용하면 24시가 넘어가는 경우를 더 간결하게 처리할 수 있다는 것을 알게 됐다.
  * **실수했던 부분 / 어려웠던 점**: 처음에는 24시가 넘는 경우를 `if`문으로만 처리하려고 해서 코드가 복잡해졌다. 총 분을 1440분(24시간)으로 나누는 아이디어를 떠올리는 데 시간이 걸렸다.

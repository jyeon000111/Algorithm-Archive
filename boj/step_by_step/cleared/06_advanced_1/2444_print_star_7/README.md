## 5. 코드 개선 (Refactoring)
- 이전 코드
```python
n = int(input())
#  출력할 모양의 너비 : 2*n - 1
# 중간 줄(n번째 줄)에서 얼마나 떨어져 있느냐를 기준으로 별의 개수가 달라진다.
# 줄은 1번째부터 (2*n -1)번째 줄까지 있다.
for row_num in range(1, 2*n):
    # n = 5 일 때,
    # 1번째 줄의 별 개수: 5 - (5 - 1)
    # 2번째 줄의 별 개수: (5 - (5 - 2)) * 2 - 1
    # 6번째 줄의 별 개수: (5 - abs(5 - 6)) * 2 - 1
    # k번째 줄의 별 개수: (n - abs(n - k))* 2 - 1
    star_count = (n - abs(n - row_num))* 2 - 1
    # (2*n - 1) 의 칸 안에 가운데 정렬한다. 빈 자리는 공백으로.
    print(f'{'*' * star_count:^{2*n - 1}}')
```

- 백준 채점 결과: '출력 형식이 잘못되었습니다' 
  - 채점 사이트에서 줄 끝에 있는 불필요한 공백을 허용하지 않는다고 함.
  - 앞에 필요한 만큼의 공백 개수를 구해서 직접 넣어주는 방식으로 변경!

- 수정한 코드
```python
n = int(input())
# 출력할 너비 : 2*n - 1
# 중간 줄(n번째 줄)에서 얼마나 떨어져 있느냐를 기준으로 별의 개수가 달라진다.
# 줄은 1번째부터 (2*n -1)번째 줄까지 있다.
for row_num in range(1, 2*n):
    # 앞쪽 공백의 개수를 구한다.
    blank_count = abs(n - row_num)
    # 해당 줄의 별의 개수를 구한다. (출력할 너비) - (2 * 앞쪽 공백의 개수) 와 같다.
    star_count = (2*n - 1) - 2 * blank_count
    # 공백 + 별 을 출력한다.
    print(' ' * blank_count + '*' * star_count)
```

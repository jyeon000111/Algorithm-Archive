## 4. 새롭게 배운 점 (What I Learned)
- XOR(Exclusive OR) 연산자 `^`: 같은 값이면 0, 다른 값이면 1 반환하는 논리 연산자
```python
x4 = x1 ^ x2 ^ x3  # 두 번 나타나는 값은 상쇄되어 0이 되고, 유일하게 한 번 나타나는 값이 남음.
```

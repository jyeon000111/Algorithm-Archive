## 4. 새롭게 배운 점 (What I Learned)
- Union-Find의 효율성은 아래 2가지가 핵심이다.
  - '경로 압축': 루트노드를 탐색하면서, 중간 다리를 바로 루트에 연결
  - 'rank 시스템': 두 집합을 합칠 때, 루트 노드의 rank(트리의 높이)가 더 높은 집합에 낮은 집합을 연결 -> 연산 횟수 줄이기

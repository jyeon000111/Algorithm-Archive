# The "Why" and "How"

## 1. 문제 정보 (Problem)

  - **출처 및 번호**: [백준 10870번]('https://www.acmicpc.net/problem/10870')
  - **난이도**: 브론즈 2
---
## 2. 핵심 아이디어 (Core Logic)

  - 재귀함수 활용
  - 정수 0 값이 들어왔을 때, 0 값을 반환한다.
  - 정수 1 값이 들어왔을 때, 1 값을 반환한다.
  - 정수 n(n >= 2)부터는 fibonacci(n - 1) + fibonacci(n - 2) 를 반환한다.

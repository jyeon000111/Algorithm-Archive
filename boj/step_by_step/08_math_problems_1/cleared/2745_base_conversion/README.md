## 1. 문제 정보 (Problem)

  - **출처 및 번호**: [백준 2745번]('https://www.acmicpc.net/problem/2745')
  - **난이도**: 브론즈 2
  - **문제 요약**: B진법 수 N을 10진법으로 바꿔서 출력

---
## 2. 핵심 아이디어 (Core Logic)
- 알파벳에 상응하는 숫자를 딕셔너리에 담는다.
- N 문자열을 뒤집어서, 인덱스를 지수로 사용해 계산한다. (enumerate 함수)

---
## 4. 새롭게 배운 점 (What I Learned)
- `int(숫자, 진법)` 숨겨진 기능
  - int 함수의 두 번째 인자로 진법을 알려주면, 알아서 10진수로 바꿔줌.
  - 이 방법은 문제에서 정해준 약속과 내장함수의 규칙이 맞아떨어질 때만 가능.
  - 문제에서의 약속이 표준적인 규범과 다른 경우도 있지 않을까?

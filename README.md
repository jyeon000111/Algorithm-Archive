## 📝 Commit Convention

#### Auto-Sync (BaekjoonHub)

> 백준허브 확장에 의해 자동으로 생성된 커밋 메시지를 따릅니다.

#### Manual Commit

> 개인적인 코드 수정 및 파일 이동 시 아래 규칙을 따릅니다.
  - `[사이트] 문제이름 / 난이도`
  - `[BOJ] 25330 거리 문자열 / 골드5`
  - `[SWEA] 1244 최대 상금 / D3`
